Module 1

HyperText Markup Language - HTML

HTML (HyperText Markup Language) is the core building block of the web. It's what gives structure to content — like headings, text, images, links, and buttons. Every website starts with HTML, because it tells browsers what to show and in what order. I began my journey here, learning how to build clean, semantic layouts.

Module 2

Cascade Styling Sheet - CSS

CSS (Cascading Style Sheets) controls the visual presentation of web content. It allows for the styling of fonts, colors, spacing, and layouts. Through techniques like Flexbox and CSS Grid, designs can be made responsive, adapting fluidly to various screen sizes and devices while maintaining visual consistency.

Module 3

JavaScript - JS

JavaScript enables dynamic behavior on websites. It allows elements on a page to respond to user interactions such as clicks, inputs, or page load events. It also provides the ability to manipulate HTML and CSS through the DOM (Document Object Model), making it possible to build real-time, interactive user experiences.

Module 4

Git & Github

Git is a version control system used to track code changes, manage project history, and enable branching for safe feature development. GitHub is a cloud-based platform for hosting repositories, facilitating collaboration, and sharing work with the development community.

Module 5

Figma & UI/UX Fundamentals

Figma is a collaborative design tool used to create wireframes, mockups, and prototypes for digital interfaces. It supports layout planning, component reuse, and design consistency. When combined with UI/UX principles, it helps produce intuitive and user-centered experiences.

Module 6

Build Systems & Bundlers

Bundlers are tools that collect and combine code, styles, and assets into optimized files that browsers can efficiently load. They help manage project structure, reduce load times, and support modular development by transforming multiple source files into a single or few production-ready bundles.
